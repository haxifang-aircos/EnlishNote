# 第一章 起点

## 我有可能把英语用的比母语还好吗

**能**。在使用多种语言时，总有一个是“主导语言”，“主导语言”是由使用频率决定的。

## “能”和“有可能”是两个概念

1 和 99 之间的差别再大，也不如 0 和 1 之间的差别大。

0 和 1 是“无”和“有”的区别，1 和 99 是积累多少的区别。没有前者，也就没有后者。

## 为什么大多数人就是学不会，甚至越学越差

因为**我们从未相信自己能够学会、学好**！

更让人不寒而栗的是，我们并不知道“我们从未相信过”。

## 什么让我们“从未相信”，甚至不知道"从未相信"

中学英语课本，绝对是罪魁祸首，起码是之一。

在我们国家，高考英语词汇姑且算它 3600 个。在高考之前，至少学习了 6 年英语，3600 个单词，每年习得 600 个英语单词，那**每天需要只要记住不到 2 个单词就够了。**

高考英语词汇在悄无声息地向你**植入**两个观点：

1. 英语很难
2. 一天能学 2 个两个单词就很了不起了

## 把植入的想法斩草除根

有一些知识是瞬间点透，并且发挥作用的。例如：

”我有可能把英语用的比母语还好吗？“

”能。“

## 与越学越差相反，并且轻松学会的人

换一种表达，“同样的环境，为什么会出现差异？”。这是**观念**的力量，可能改变一个人的一生。

## 自证预言，强大的诅咒

相信自己能学好英语，并不一定会学好；不相信自己能学好，一定不会学好。

这与人类大脑中根深蒂固的 “恐惧情绪” 心理有关。

## “学”好了，才能“用”？

这个假定，在很多情况下是成立的。医生、律师、开车等。

没“学”好，“用”英语不会伤害任何人。

## 英语对你的作用

English, for me, freedom.

## 总结

1. 别对自己的要求太低了
2. 足够多的输入，才能带来输出
3. 做事的时候，出错是必然的，如果正在做事却一点错都没有，那不是做事──那是在做梦
