# 起点

## 我有可能把英语用的比母语还好吗

**能**。在使用多种语言时，总有一个是“主导语言”，“主导语言”是由使用频率决定的。

## “能”和“有可能”是两个概念

1 和 99 之间的差别再大，也不如 0 和 1 之间的差别大。

0 和 1 是“无”和“有”的区别，1 和 99 是积累多少的区别。没有前者，也就没有后者。

## 为什么大多数人就是学不会，甚至越学越差

因为**我们从未相信自己能够学会、学好**！

更让人不寒而栗的是，我们并不知道“我们从未相信过”。

## 什么让我们“从未相信”

中学英语课本，绝对是罪魁祸首，起码是之一。
